# 课程笔记智能整理系统 - 使用指南

## 🎯 已实现功能（第一阶段）

### ✅ 核心功能
1. **多格式资料处理**
   - 音频转文字（Whisper）
   - 图片OCR识别（PaddleOCR）
   - 文档提取（PDF/PPTX/TXT）

2. **教师提纲集成**
   - 在界面输入教师提纲和重点
   - AI 自动按提纲结构组织笔记
   - 思维导图体现提纲层级

3. **数据库持久化**
   - 所有课程、提纲、会话保存到 SQLite
   - 历史记录永久保存
   - 支持按课程名检索历史

4. **在线编辑器**
   - Markdown 编辑器（EasyMDE）
   - 工具栏：标红/高亮/批注
   - 实时预览
   - 思维导图可视化

5. **三视图切换**
   - 📝 编辑笔记
   - 👁️ 预览
   - 🗺️ 思维导图

---

## 🚀 使用流程

### 第一步：上传资料
1. 拖拽或点击上传课堂文件（音频/图片/文档）
2. 输入课程名称（如：高等数学-第三章）

### 第二步：输入提纲（可选）
在提纲输入框中输入：
```
1. 极限的定义（重点）
2. 连续性判定
3. 间断点分类（必考）
```

### 第三步：开始整理
点击「🚀 开始整理」，等待处理完成

### 第四步：编辑和标注
- 使用工具栏标红重点
- 添加批注
- 切换到预览查看效果

### 第五步：下载文件
- Markdown 笔记
- Word 文档
- HTML 文件

## 📊 数据库说明

系统在项目目录下创建 `knowledge.db`，包含以下数据：

- **课程信息**：课程名、学期、老师
- **教师提纲**：提纲内容、章节树
- **整理会话**：每次整理的完整记录
- **历史追踪**：可追溯所有历史版本

## 📊 完整数据流程

```
【上传并整理】
用户上传文件 + 点击"开始整理"
    ↓
保存课程名到 localStorage
    ↓
调用 /process API 上传文件
    ↓
pollStatus() 每秒轮询处理进度
    ↓
收到 { status: 'done', notes_md: '...', mindmap: '...' }
    ↓
displayResults(data)
    ├─ displayKnowledge(notes_md) → 渲染知识点卡片
    ├─ displayMindmap(mindmap) → 渲染思维导图
    └─ switchTab('knowledge') → 自动切换到知识总结
    ↓
loadHistory() → 刷新历史记录列表
    ↓
✅ 用户看到完整结果，无空白页
```

```
【页面初始化】
用户打开/刷新页面
    ↓
loadLatestResult() 自动执行
    ↓
从 localStorage 读取 lastCourse
    ↓
调用 /api/courses/{course}/history
    ↓
displayHistory() → 显示历史列表
    ↓
loadHistoryItem(最新记录ID)
    ├─ displayKnowledge() → 显示知识点
    ├─ displayMindmap() → 显示思维导图
    └─ switchTab('knowledge') → 切换到预览页
    ↓
✅ 页面立即显示上次的整理结果
```
